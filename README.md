# Bầu cử ĐBQH và HĐND các cấp - Phường Ayun Pa

Đây là bộ source code hoàn chỉnh đến thời điểm hiện tại.

## Điểm nổi bật
- Phân quyền theo khu vực bỏ phiếu
- Nhập tổng số cử tri và số đã bỏ phiếu hiện tại
- Mốc báo cáo 8h, 10h30, 14h, 18h, 21h
- Trang thống kê khu vực chưa báo cáo theo mốc thời gian
- Xem báo cáo, tổng hợp, xuất Word và Excel
- Chân trang bản quyền:
  Ủy ban bầu cử phường Ayun Pa
