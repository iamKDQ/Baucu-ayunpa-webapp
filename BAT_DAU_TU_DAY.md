# BẮT ĐẦU TỪ ĐÂY

Tài liệu này dành cho người chưa biết gì về công nghệ thông tin.

Bạn chỉ cần làm đúng thứ tự dưới đây.

## Mục tiêu
Sau khi làm xong, bạn sẽ có:
- 1 website chạy được trên Internet
- 1 tài khoản quản trị để đăng nhập
- 1 cơ sở dữ liệu để lưu tiến độ bầu cử
- 1 dashboard công khai
- 1 khu vực quản trị
- 1 chức năng xuất báo cáo Word

## Nếu bạn rất mới
Hãy làm đúng quy trình này:
1. Cài phần mềm vào máy
2. Chạy thử app trên máy tính của mình
3. Tạo Supabase
4. Tạo tài khoản admin
5. Đăng nhập thử trên máy
6. Đưa source code lên GitHub
7. Kết nối GitHub với Netlify
8. Khai báo biến môi trường trên Netlify
9. Deploy lên Internet
10. Kiểm tra website hoạt động
